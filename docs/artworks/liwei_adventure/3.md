# 李伟的奇妙冒险--第三章

目前李伟等人也很忙。李伟拿起了乌乌里大黑杆，然后打开了"靴消炎大战王建树114514回合.mp4"。没想到她不小心执行了"CHC'sDeterrence.EXE"和"a.bat"电脑放出了CHC的审判曲"Chemical lov"的精污＋扭曲＋变调＋Remix版，同时文件开始发射当前位置、时间等信息。没过多久大四中帝国就来敲门了。

大四中帝国：敲敲门，美利坚来咯。

李伟没有管他们。结果她的电脑被强制进入了"CHC审判战game.c"，计时器上显示了15min,这意味着如果李伟没有在15分钟内通关，她的电脑就要白给了哦。

笑死，这个游戏中CHC对话就花了1分钟，紧接着﹣﹣游戏开始！

CHC：多么美好的一天啊，小花在歌唱，小鸟在绽放，在这样的一天里，像你这样的人﹣-

CHC（黑眼）: 应当在硝酸里焚烧

李伟：卧槽卧槽，这屑CHC竟然还有龙骨炮。

菜的一批的李伟在第一关就逝了。最侮辱人的是，CHC播放了"李伟威慑失败结算动画．mp4"和"GameOver.jpt"，李伟就读档了一次。

CHC：哇哦，这是你来的第仁次。

结果这一会李伟仍然没有过开头杀。

CHC：第仨次来这里了耶。

这会李伟看见了"是否降低难度？"李伟按了一下"是"，结果又弹出了"确定吗？这是给骷髅宝宝玩的"

结果李伟惊喜的发现，降低难度后，她连第一个攻击都挨不了。

CHC：这是你的第伵次。让房间冷下来吧，让鲜血飞溅吧，你这肮脏的自由杀手。

这回李伟总算过了开头杀。李伟选择"Fight"，结果CHC闪避了李伟的攻击。

李伟：卧槽这个屑还会闪避。

CHC：我看你是没逝找抽。

CHC使出了针头攻击。李伟竟然用跳跳跳躲了过去。不过第三回合李伟就没那么好运了，她不知道在蓝色攻击时要停止，结果李伟的HP只剩1了。

结果因为李伟没有吃补给，第四回合是被一连串剑斩揍死，白给了。

CHC：哗哗，这将是你的第伍次逝世。

不出CHC所料，李伟直接被大烧瓶轰死了。

不过李伟在第六次时欧气爆棚，直接打到了第一阶段的倒数第二个回合。

CHC：这一回合我不想攻击，我将要让鳝丝来帮我。

只见屏幕出现了一个鳝丝罐头，上面写着钐斯。李伟竟然躲过了带有钐同位素的放射钐骨炮

追杀，然后李伟一杆把鳝丝打跑了。

CHC：你竟然打鳝丝大跌，你可真是令人作呕啊。

结果李伟在下一回合被飞刀击中了。

到了第八次，李伟总算过了第一阶段，可是……正常来说，通关CHC战需要10分钟，而李伟已经超时了。

于是李伟的电脑黑屏了。

李伟重启了电脑，一切很正常，但是"CHC审判战game"却依然在电脑上。李伟想要把它拖到垃圾桶，结果"CHC审判战game"却强制打开了。

李伟：我日你娘！

李伟只好不管它。她打开了对原版乌乌里大黑杆的解析图。

这会她打算搞一些鬼才主意。她搞到了一些高强度物质（当然，它们价格不菲）、镍合金，以及一些钨钢。经过长时间的熔炼、铸造、打磨后，制造成了一种含有强内应力物质的，极为光滑而坚硬的乌乌里大黑杆。

她还从网上买来了大功率电动推进器，放在乌乌里大黑杆的尾部，同时安装上一些尾翼，并在尾翼上安装了推进器。

李伟干完这些后，就前假期托管，好家伙，一个也没有，倒是815班有些喧哗。李伟：呕吼吼，看了还得是你，815班……卧槽这门怎么打不开？

李伟发现地上有一个踩压力板的谜题与1个告示牌。李伟先是踩上一个压力板，然后跑着去踩另一个，结过压根踩不上3个。

告示牌：是不是怎么也踩不上？试着从面积为4√3的正三角形的边长开始倒序踩吧！

李伟：卧槽这是甚么鬼玩意儿……

李伟选择咩力大仙。咩力大仙列了1个一元二次方程，为（(√(x^2-(x/2)^2))x)/2=4/3。结果李伟取了﹣4这个实数根。没想到智商不好的李伟歪打正着，以﹣4,-3,-2,-1的顺序踩了压力板，门打开了，然后给了李伟一个大逼斗。

接着李伟遇到了一个10x10踩雷，外加一个告示牌。

告示牌：这个谜题是200％反入类的！（后面写着狗爬字）

李伟俩眼一黑，这个踩雷游戏底下放的是真的雷，也就是说，李伟那种不命大的人必须要一命通关。（别问815教室里的人是怎么进来的，他们个个都是"真．顶尖人物")

李伟突然想到一个作弊妙招，她想起来CHC每次的爆炸手段都是使用像雷酸汞那样的可溶爆炸物。于是李伟拎来一桶水然后踩了一个3，打算溶解掉雷酸汞与雷酸银。结果水非但没有完全溶解爆炸物，反而冲落了隔板，造成连锁爆炸。更要命的是，这些爆炸物里还有气体（如甲烷和乙炔）和碱金属……

Bonnmmm!

如此大的声响，里面的人肯定听见了。

喧哗声炸燃而止。然后，里面传出一声……

『 所有攻击单位，瞄准目标，开火！』

Weeu--BonnBonnBonnBonnBonn--

李伟：卧 槽

李伟在子弹中穿梭。有好几次，子弹差点命中要害，呼啸着飞来飞去。李伟的衣服被打烂了，幸好里面有一件足够厚而有韧性的毛衣，不然惨不忍睹。

忽然，弹雨停了，只有一个直径4.5mm枪炮管对着李伟。然后，那位似乎扣动了发射器，连射10发，全都爆头了。

SRAO成员：嗯哼哼，十发都中了。

CHC：枪枪命中，全靠这台按（氨）瘾精灵。

打中了，而且了李伟的脸。李伟狼狈不堪的跑走，结果撞上了一张关于锘钡鿭的报纸，报纸后面正是走来的刘凯丽！

李伟：刘凯丽老师，赶紧去管管815班吧，都乱成亿锅粥了！

刘凯丽就朝着815班走过去。利用"拍森"的编程，她走过第一个谜题。然后第二个谜题就没那么简单了。前三个砖就把她住：6//4+1(-1)2223、[aretan(0.5)+arctan(1/3)]÷10-1.5与（0!+0!+0!)!=3。

刘凯丽的手机都没电了，她只能自己算数。但是她懒得算。于是她就一直往前冲击，企图用高速的动能撞烂谜题门。

Weeu--Bonnmmm!

刘凯丽一路踩雷，最后撞到了紧闭的门，发出一声巨响。几乎在同时，所有攻击单位火力全开，但就是打不到刘凯丽。因为门口那里是一个攻击死角。刘凯丽看着打不到她，就尝试用各种方法开门，又是肘击又是用拳头砸，但是就是打不开门。但是她成功让里面的人发现了她的位置。

SRAO成员1：报告，刘凯丽在前门！

马兰：主权不整，班级必亡！同志们，启动！!!

屋里的入停止了扫射。只见刘凯丽头上的一个挡板没了，从挡板里伸出来1只拳头，直接朝着刘凯丽脑门上砸去。刘凯丽被砸得眼冒金星，紧接着又来一拳。

刘凯丽还挺识相，趁着SRAO成员们没有注意，从他们的攻击范围内溜走了。

刘凯丽：留得青山在，不怕没柴烧，你们走着瞧！

CHC：你们平白无故打扰我们，我看你没逝找抽！我踏马不氰死你！准备，放氰气！

但是刘凯丽已经走了，就算放氰气也没用。不过李伟这会把昌盛给带过来，想要侵入815班。结果昌盛刚刚开门，浓烈的氰气就传来了。昌盛老登一看不妙跑了，只留下李伟，被氰气熏了个半死不活。

李伟：你竟然连你们班的学生都不管，你＊*

高海伦拿到全新乌乌里大黑杆后赞不绝口，他仔细看了看这根杆子，又拿它挥舞了几下，感觉比原版和竹竿要好多了。

高海伦：竟然是强内应力材料，这可是无论如何也不会破碎的啊，比钛合金好得多。他赶忙命令学生会给他制造。但是他犯了一个极大的错误，就是用廉价的铝代替了部分钨钢，甚至有一根乌乌里大黑杆全是由铝合金做的。这压根称不上强内应力材料。不过勉强应该可以。毕竟铝合金的强度很高（但是耐腐蚀性是个问题）。

高海伦还在乌乌里大黑杆上安装了定位器。现在，乌乌里大黑杆已经成为了一种极为强大与恐怖的存在。不过再强大的东西也有弱点，由强内应力材料制成的乌乌里大黑杆也不例外。乌乌里大黑杆2.0和Zongsketball20.2究竟谁更胜一筹？请拭目以待。
