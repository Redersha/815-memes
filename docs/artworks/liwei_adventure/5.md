# 李伟的奇妙冒险--第五章

马兰最后如愿以偿的登上了满护甲的"山鸡号"。与他同行的还有王建树与一个小助手。

马兰：CHC你的强内应力护甲不错啊，向水晶一样。

CHC：......

马兰：CHC你是魔怔了吗，怎么不说话？

CHC：你可能不知道，强内应力材料都有一个极为致命的弱点，不管是乌乌里大黑杆2.0还是强内应力护甲。我忘记了跟你说这事……

马兰：没事，无非就是砸一个大坑呗，我在外面有加上了一层钛合金防护障，隔层也抽真空了，不用怕他们。

CHC：那么，祝你好运。对了，当心乌乌里大黑杆内部攻击！

只可惜，马兰没有听到这最后一句话。他把一包"马兰"种子与一个宗球揣兜里就走了。目前所有的战舰与搜救舰都停留在了操场上的停舰港。学生们蓄锐待发，而现在，马兰会见了"适则生存号"的舰长﹣﹣晓北。

晓北：久仰久仰，我早已经听说了鸡咩译牢上将的大名。

马兰：（你认真的吗？）哈，谢谢夸赞。其实我也就是一个普普通通的科学员，这次登舰考察李伟的新武器﹣﹣乌乌里大黑杆2.0。还请多多配合。

晓北的脸一听到"乌乌里大黑杆2.0"就愣住了。但是他很快以难以发现的速度恢复了原貌。之后，他们自登自舰了。

李伟、高海伦与学生会的新舰队，正在另一头集结。他们的任务很明确：消灭学生舰队，镇压进行起义与革命的学生，抹杀学生们的自由，将他们由实实在在的人驯化为顺从的绵羊！李伟：高海伦，你这次可是建立的大功，制作了8根乌乌里大黑杆！

高海伦：在这场战役结束后，我们还可以用这些杆子抽打那些学生！

体育老登郭晓毁：到时候我们就可以控制那些学生了，我们这会子也不用怕他们的什么球了！众老登：哈哈哈哈哈哈……

不过，这会李伟的布局却有点大大的奇怪，不仅在学生方面的意料之外，甚至还在他们自己人的意料之外，连李伟的人机和学生会首领都没有料到。

只见李伟的舰队弧状排开，大部分战舰在后方，只在前方留了九个战舰。

双方经过一个多月的精心准备，实验末日战役打响了。此时，学生舰队的"新世界号"发生了一些骚动。

马敏杰：浩泽，我让你加20％的氨气、20％的酒精与5％的液氮，你怎么加了25％的液氮和20％的酒精？那样我们的战舰恐怕会炸掉的。

浩泽：啊？我一直呆在这里远程操控啊。会不会是有人盗用了权限，加了那么多的液氮？

马敏杰：行吧，那么我们去看看是不是有人在那里装神弄鬼。下次记得去动力舱亲自操控。

马敏杰与浩泽去了动力舱，里面却空荡荡的，一个人也没有。马敏杰用最高权限退掉了多余的液氮，加上了氨气，并且锁定了燃料。他们随后来到了驾驶舱。发现有一个船员正在操控着动力系统。

马敏杰：不许动！ 

船员：不不不，我，我真的是在为你们好！

浩泽：你这已经属于叛逃行为，需要被CHC的热武器崩掉！

船员：你们先听我解释！

马敏杰：好吧，我听你解释。浩泽，你回动力舱，让我先看看这个船员怎么解释，说不定有点道理。

船员：是...是这样的！李伟的那个新型武器，任何人都无法捕捉！别说是"山鸡号"，就连"红镰号""重锤号""石英﹣5""正义号"都没法捕捉它！它的威力无穷，任何碰到它的东西都会瞬间支离破碎！我们……我们最好的方法就是快点逃避！我想，刚才浩泽副舰长的话不太对……我们是逃跑了，但是，我们至少没有背叛！

马敏杰：你说得对。但是你加那么多液氮干什么，那样会导致爆炸。

船员：我们需要尽可能快速逃离。只要快点跑一段距离，舰队班际就不会追我们了！

马敏杰：主意不错。但是主控制权在我手里，最高权限也在我手里。你应该让开点，由你指

挥逃跑，我来操纵。

船员为马敏杰让开了通往操作台的道。马敏杰并没有去操纵，而是扑向了那个船员。那个船员被五花大绑控住后依然不死心，他从袖口里掏出一枚抛射式碱龙息弹，然后拉开了它。说时迟，那时快。马敏杰一下夺走了龙息弹，然后一下把它扔进了附近的水池里。

Puff--Bonnmmm！水池直接爆炸出了一个巨大的蘑菇云。（碱金属与水产生剧烈反应）此时浩泽刚刚带着三个人从动力舱里过来，目睹了这一切。他们成功的把这个钢印族船员制服了。

浩泽：老大帅气！老大威武！

马敏杰：别吹了。赶紧把这个船员送到那个楼调查！

黑灰相间的门后面的会议房间，STD主席机构正在与最后两个面臂者谈话。

STD主席：自面臂计划开始的几个月以来，所有面臂者都进行了对抵抗李伟的多次尝试。但是效果并未达到预期，反倒浪费了许多资源。现在，我们将对终止面臂法案与面臂计划。STD主席：同意者，举手！

在会的13人，8人同意，5人反对。

STD主席：既然有5人反对，超过同意数的一半，那么们决定只废除第三位面臂者的面臂权利。第三位面臂者，请问，你是否同意？

第三位面臂者莞尔一笑。

第三位面臂者：既然是主席与大多数人的要求，我同意废除我的面臂权利。

彬彬（面臂者﹣4):（这好事怎么没落我头上呢？)

此时，第三位面臂者的同伴却急着要求发言。

STD主席：对不起，你不是面臂者，你无权发言。

破臂者﹣3：不，你不知道。第三位面臂者，我是你的破臂人。

破臂者﹣3：他看似只造了1个脑印，但是实际上，脑印不只有1个，他一共制造了5台脑印。然后秘密的把它们交给了钢印族。如果它们保养得当并且循环轮流使用，完全可以撑到现在。

破臂者﹣3：而且，在脑印的代码里，他加了一个小小的负号，就连李伟的探测器也没有发现，这直接逆转了原来的命题！

STD副主席：这意味着什么呢？

破臂者﹣3：这意味着……

舰队班际：非法离港！ASNF发生了非法离港！

SRAO司令员1：什么情况？

SRAO司令员2：非法离港，"适者生存号"跑了！可恶，他们使用了免密。

SRAO司令员3：快点上报数据，麻溜的！

SRAO总司令在他的操作室打着数据，而马兰透过"山鸡号"的舷窗，一脸懵逼的看着"适者生存号"划过天际。

CHC在"红镰号"上担任指挥官。他一抬头就也看见了"适者生存号"。

马兰：我祝它早日坠毁……有点意思……太狡猾了。

STD副主席：第三位面臂者先生，看你干的好逝！

第三位面臂者（莞尔一笑）：我认为，如果我们硬拼我们没有任何一点胜算。不如保存实力，向更好的远方前进。

STD全员无话可说。突然，有人说了一句，打破了寂静。

议员1：第四位面臂者搁哪去了？

议员2：他？他早跑了。

彬彬已经回到了房间。白色的大显示屏上空无一物。彬彬就一边吃华莱士一边换挡，换

为"CHC新闻"。画面上出现了"红镰号"，但是没有CHC，取而代之的是SRAO总司令。

此时他正在播报关于"适者生存号"的逃跑与"麦芒号"被战前袭击是消息。

SRAO总司令：我们可以看见"适者生存号"已经逃跑。目前，我们已经派出了"毅力号"竞争号""究极法则号"与"新世界号"前去追击！

SRAO总司令：我们肯定，在我们的舰队中，肯定还有很多的想逃跑的钢印族，但是我们会一步一步的踢掉他们！

SRAO总司令：我们知道，他们虽然逃跑，但是没有背叛，但是临阵脱逃是一个恶劣的行为。

SRAO总司令：因为：即使我们尽全力去打但被打败，那就是虽败犹荣！

这时，ASNF舰队发出号令：山鸡号，出列！
